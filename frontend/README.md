# CreditGuard Frontend

React + TypeScript UI for fraud detection.

## Quick Start

```bash
npm install
npm run dev
```

App: http://localhost:5173

## Components

- `App.tsx` - Main layout
- `TransactionForm.tsx` - Input form
- `ResultCard.tsx` - Results display
- `api.ts` - Backend client
- `types.ts` - TypeScript interfaces

## Build

```bash
npm run build
```
